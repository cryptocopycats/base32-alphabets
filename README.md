# Base32 Encoding / Decoding - 5-Bit Kai Notation/Alphabet

Encode / decode numbers in 5-bit groups (2^5=32)
with the kai notation/alphabet (1,2,3,4,5,6,7,8,9,a,b,c,d,f,g,h,i,..)

Note: Following Base56 - the kai alphabet
uses no 0 and l ...
